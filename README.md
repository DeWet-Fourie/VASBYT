# VASBYT Overland V4 — Bold Offroad Catalogue

A premium WhatsApp-first outdoor/4x4 product catalogue prototype.

## What changed in V4

- Cluttered top navigation removed.
- All pages now use a single hamburger/accordion drawer menu.
- Visual direction changed to a bold offroad campaign style: large condensed headings, orange accent blocks, dark navy sections, strong image panels and higher-contrast product/collection cards.
- Collections are still controlled from `data/store-data.js`.
- Vague search/recommendation system remains active on the shop page.
- Footer includes support and legal links, including Terms & Conditions, Privacy and Returns.

## Main editable file

```txt
data/store-data.js
```

Edit:

- `STORE_CONFIG`
- `MENU_COLLECTIONS`
- `PRODUCTS`

## Upload to GitHub

Upload the contents of this folder into the repository root:

```txt
index.html
shop.html
product.html
about.html
contact.html
terms.html
assets/
data/
docs/
README.md
```

Do not upload the wrapper folder itself.
