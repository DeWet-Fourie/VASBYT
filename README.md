# VASBYT Overland V5

Clean presentation build.

## What changed
- Shorter homepage: hero, collections, featured gear, footer only.
- Removed clutter and the extra sales-system section.
- Removed Home from the menu drawer. The logo still links home.
- Cleaner shop finder section.
- Improved WhatsApp quote controls so quantity buttons do not overlap.
- Footer keeps Terms & Conditions, Privacy Policy and Returns Policy links.

## Deploy
Upload the contents of this folder to GitHub Pages or Netlify. Do not upload the wrapper folder itself.

## Main editable data
`data/store-data.js` controls products, collections, menu order and contact settings.
